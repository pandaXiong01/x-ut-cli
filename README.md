# x-ut-cli
前端单元测试脚手架，方便老项目集成单元测试框架

## 背景
准备对公司一堆项目集成unit test，为了方便团队统一接入，做了个命令行工具。

## 使用说明

```bash
x-ut-cli init
```
选择项目的单元测试配置，有下面三种选择。
- javascript + jest 
- vue + vue-test-utils 
- react + react testing library 



### 命令

- 初始化或升级项目配置

```bash
x-ut-cli i
```
